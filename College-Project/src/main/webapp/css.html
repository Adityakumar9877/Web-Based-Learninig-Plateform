Selector

  1. Introduction
    • It is used to target elements, so that we can apply property.
  2. Simple Selector
    • Tag Selector:- To target element using tag selector we have to use tagname.
    • syntax:-
    • Tagname { property:value;}
    • Id selector:-To target element using id selector we have to use hash('#) before the id name.
    • Syntax:-#id_name { property:value; }
    • Universal Selector:- It is used to target all Html element in webpage.It's symbol is asterik-(*)
    • syntax:- * { property:value; }
    • Grouping Selector:-To target multiple elements we can use grouping selector and to group selector, simply separate each selector with a comma(,)
    • syntax:- Selector,Selector2 { property:value; }
      • 3.Combinator Selector
      • Descendant Child Selector:-to target all Direct childs and indirect childs we have to use space after the parent selector
      • syntax:-parent child { property:value; }
      • 4.Attriute Selector:-
      • By Attribute Selector :- we can target elements based on the attribute and their values.
      • 5.Pseudo Class Selector :- To target elements on the basis of their state.Pseudo-classes are written with a colon(:) followed by the name of the pseudo-class.
      • 6. Pseudo Element Selector :- To target specific part of an element.They are denoted by a double colon(::) before the name of the pseudo-element.
