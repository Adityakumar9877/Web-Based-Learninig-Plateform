What is HTML5?

HTML5 is the latest version of HTML, designed to structure and present content on the web efficiently. It supports multimedia, graphics, APIs, and semantic elements.

What are semantic elements in HTML5?

Semantic elements clearly describe their meaning to both the browser and developer, like <header>, <footer>, <article>, <section>.

What is the purpose of the canvas element?

The <canvas> tag is used to draw graphics dynamically using JavaScript, such as graphs, animations, or game visuals.

What is the use of the audio tag?

The <audio> tag embeds sound content, like music or podcasts, into a web page without external plugins.

What is the use of the video tag?

The <video> tag embeds video content such as movies or video streams directly in HTML pages.

What is localStorage in HTML5?

localStorage allows web applications to store data in the browser with no expiration date, persisting even after the browser is closed.

What is sessionStorage in HTML5?

sessionStorage stores data only for the duration of the page session and is cleared when the browser tab is closed.

What is the difference between localStorage and sessionStorage?

localStorage persists data across sessions, while sessionStorage clears data when the tab is closed.

What is the purpose of the meta charset tag?

The meta charset tag specifies the character encoding for the document, ensuring text is displayed correctly in all browsers.

What is the use of the progress tag?

The <progress> tag represents the completion progress of a task, like a file download or form submission.

What is the purpose of the meter tag?

The <meter> tag shows a scalar measurement within a known range, like disk usage or performance metrics.

What is the draggable attribute?

The draggable attribute allows an element to be moved using the Drag and Drop API.

What is the purpose of the required attribute?

The required attribute ensures a form field must be filled before form submission.

What is the placeholder attribute?

The placeholder attribute provides a short hint describing the expected input in a form field.

What is the purpose of the data-* attribute?

The data-* attribute is used to store custom data private to the page or application.

What is the hidden attribute?

The hidden attribute hides an element from the page but it remains in the DOM.

What are Web Workers?

Web Workers allow JavaScript to run in the background without blocking the UI.

What is Geolocation API?

The Geolocation API allows websites to access the user’s geographic location with permission.

What is the purpose of the iframe tag?

The <iframe> tag embeds another HTML page within the current page.
