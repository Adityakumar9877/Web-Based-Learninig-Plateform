What is CSS?

CSS (Cascading Style Sheets) is used to style and layout HTML elements on a web page.

What are the different types of CSS?

Inline CSS, Internal CSS (inside <style>), and External CSS (linked via <link>).

What is the difference between relative, absolute, fixed, and sticky positioning?

CSS positioning defines how elements are placed. Relative moves relative to its normal position, absolute moves relative to nearest positioned ancestor, fixed stays relative to viewport, and sticky toggles between relative and fixed.

What is the difference between id and class in CSS?

An id is unique and used once per page (#id), while a class can be used multiple times (.class).

What are pseudo-classes?

Pseudo-classes define the state of an element, such as :hover, :focus, :first-child, :nth-child(n).

What are pseudo-elements?

Pseudo-elements style specific parts of an element, like ::before, ::after, ::first-letter, ::first-line.

What is the difference between relative units (em, rem, %) and absolute units (px, cm)?

Relative units adjust based on parent or root element, while absolute units are fixed size regardless of parent.

What is the difference between inline, inline-block, and block elements?

Block elements take full width and start on a new line, inline elements stay in line, inline-block behaves like inline but allows width/height.

What is the difference between visibility: hidden and display: none?

visibility: hidden hides the element but keeps space, display: none removes the element and space.

What is the difference between relative, absolute, and fixed background positioning?

background-position determines the starting point of the background image relative to the element, page, or viewport depending on CSS property values.

What is the difference between inline styles, internal CSS, and external CSS?

Inline styles apply directly to an element, internal CSS is inside <style> in the page head, external CSS is in a separate file linked via <link>.

What is the difference between relative and absolute URLs in CSS?

Relative URLs point to resources relative to the CSS file location, absolute URLs include the full path including domain.

What are media queries?

Media queries allow CSS to be applied conditionally based on screen size, resolution, or device type for responsive design.

What is the difference between inline, internal, and external CSS specificity?

Inline styles have highest specificity, then internal styles, and external styles have lowest unless overridden.

What is the difference between relative, absolute, fixed, and sticky positioning?

Relative is relative to original position, absolute relative to nearest positioned ancestor, fixed relative to viewport, sticky switches between relative and fixed based on scroll.

What is the difference between inline-block and block?

Block elements occupy full width and start on a new line, inline-block elements do not start on a new line but can have width and height set.

What are CSS transitions?

CSS transitions define smooth animations for property changes over a duration when state changes, e.g., hover effects.

What are CSS animations?

CSS animations allow multiple keyframe-based animations on an element with control over duration, iteration, and timing functions.

What is the difference between relative, fixed, and sticky position in CSS?

Relative is offset from normal position, fixed stays relative to viewport, sticky switches between relative and fixed when scrolling.

What is the difference between absolute, fixed, and sticky background positioning in CSS?

Absolute positions relative to element, fixed positions relative to viewport, sticky toggles between relative and fixed based on scroll.
