Java – Complete Learning Notes

These Java notes are designed for beginners, students, and aspiring software developers. They are suitable for learning platforms, college notes, and interview preparation.

1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It is widely used for web applications, mobile apps, desktop software, and enterprise systems.

Java follows the principle: Write Once, Run Anywhere.

2. Features of Java

  • Object-Oriented
  • Platform Independent
  • Secure and Robust
  • Multithreaded
  • High Performance

3. Java Architecture

Java programs are compiled into bytecode. This bytecode is executed by the Java Virtual Machine (JVM), which makes Java platform-independent.

  • JDK – Java Development Kit
  • JRE – Java Runtime Environment
  • JVM – Java Virtual Machine

4. Variables in Java

Variables are used to store data values. Java supports different types of variables.

  • Local Variables
  • Instance Variables
  • Static Variables

5. Data Types in Java

  • Primitive: int, float, double, char, boolean
  • Non-Primitive: String, Array, Class, Interface

6. Object-Oriented Concepts

  • Class
  • Object
  • Inheritance
  • Polymorphism
  • Abstraction
  • Encapsulation

7. Java Example Program

Simple Java program to print Hello World:

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

8. Advantages of Java

  • Large community support
  • Used in enterprise applications
  • Secure and reliable
  • High demand in jobs

9. Conclusion

Java is one of the most popular programming languages. Learning Java opens doors to careers in software development, backend development, Android development, and enterprise systems.
